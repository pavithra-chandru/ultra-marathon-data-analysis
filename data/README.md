# Dataset

The dataset used in this project is **not included** in this repository because of its large size.

## Dataset Source

Kaggle – The Big Dataset of Ultra Marathon Running

https://www.kaggle.com/datasets/aiaiaidavid/the-big-dataset-of-ultra-marathon-running

## Download Instructions

1. Visit the Kaggle dataset page.
2. Download the dataset.
3. Extract the CSV file.
4. Place the CSV file inside this `data/` folder before running the notebook.
